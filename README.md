# Raspberry PI5 bash scripts

this repo contain a few bash scripts to help you configure your raspberry pi5

## N8N with Ollama and Phi3 model configure script

```bash
./configure_n8n.sh
```

## 5GH hotspot configure script

```bash
./configure_hotspot.sh
```

## Vpn routing configure script

```bash
./config-vpn-routing.sh
```

## Filter domains script

```bash
./filter_domains.sh
```

## Openclaw configure script run phi3 model in ollama

```bash
./configure_openclaw.sh
```
